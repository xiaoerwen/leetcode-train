# 231. 2的幂

给定一个整数，写一个函数来判断它是否是 2 的幂次方。如果是，返回 true ；否则，返回 false 。

整数 n 是 2 的幂次方需满足：存在整数 x 使得 n == 2^x。

## 示例 1

```
输入：n = 1
输出：true
```

## 示例 2

```
输入：n = 16
输出：true
```

## 示例 3

```
输入：n = 3
输出：false
```

## 示例 4

```
输入：n = 4
输出：true
```

## 示例 5

```
输入：n = 5
输出：false
```

### 提示

- -2^31 <= n <= 2^31 - 1
